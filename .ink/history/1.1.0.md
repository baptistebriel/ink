# 1.1.0

> 2026-03-31 — feat

## What changed

Scaffold now generates agent instructions for all major AI coding tools:

- `CLAUDE.md` — Claude Code (auto-loaded at session start)
- `AGENTS.md` — Cursor, Cline, pi, Aider, Continue (cross-tool standard, 20+ tools)
- `.github/copilot-instructions.md` — GitHub Copilot

All three files contain identical instructions, generated from the same template.

## Why

`AGENTS.md` alone wasn't being auto-loaded by Claude Code (which reads `CLAUDE.md`) or GitHub Copilot (which reads `.github/copilot-instructions.md`). Each tool has its own convention — ink now covers them all so agent instructions work regardless of which tool you use.
