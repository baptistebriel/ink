# 1.1.0

> 2026-03-31 — feat

## What changed

Scaffold now generates both `CLAUDE.md` and `AGENTS.md` for cross-tool compatibility.

- Claude Code reads `CLAUDE.md` by default — without it, agent instructions were silently ignored
- Other tools (Cursor, pi, etc.) read `AGENTS.md`
- Both files contain identical instructions, generated from the same template

## Why

`AGENTS.md` alone wasn't being auto-loaded by Claude Code, which only reads `CLAUDE.md`. This meant agents using Claude Code had no awareness of ink's commit rules or version workflow unless manually told to read the file.
